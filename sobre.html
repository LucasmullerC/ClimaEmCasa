Sobre

ClimaEmCasa é um site que traz um simples serviço, mas que ainda é importante, de verificar o clima em uma determinada localidade basta somente inserir o nome da sua cidade que em instantes você saberá se vai poder ir para a praia ou se vai ter que usar um casaco no final da tarde.

🌎 Visão Geral

O ClimaEmCasa é um projeto feito por três estudantes para a disciplina de Programação Web e Mobile da Universidade Católica De Pernambuco. O projeto foi visado para ser feito de uma forma simples e com um belo visual, utilizando no mínimo uma API Rest.

🛠 Tecnologias

As seguintes ferramentas e APIs que foram usadas na construção do projeto:

  1. Bootstrap
  2. OpenLayers
  3. Open Weather Map
